## 📍 Overview

`oco` is a CLI option processor. It lets you parse and manipulate the values of
CLI options before pushing them to another command.

---

## 🚀 Getting Started

### 📦 Installation

```sh
cargo install oco
```

### 🎮 Using oco

```sh
oco --help
```
